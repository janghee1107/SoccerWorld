<h1>SoccerWorld</h1>
안녕하세요 축구를 좋아하는 누구나 이용할 수 있는 웹사이트로<br>
잉글랜드 프리미어리그(EPL), 라리가, 세리에 A, 분데스리가의 경기 일정 및 순위 확인과 
승부 예측이 가능합니다.<br>
또한 커뮤니티 기능을 통해 사용자 간 자유로운 소통이 가능합니다.
